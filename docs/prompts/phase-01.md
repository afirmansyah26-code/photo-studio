# Objective

Build the complete application shell.

Focus only on the UI foundation.

No image processing should exist in this phase.

---

# Scope

Implement:

- App Layout
- Header
- Sidebar / Toolbar
- Workspace
- Preview Panel
- Footer
- Responsive Layout
- Light Theme
- Dark Theme
- Design Tokens
- Basic UI Components

---

# Do NOT Implement

Do **NOT** implement:

- Upload
- Drag & Drop
- Crop
- Rotate
- Zoom
- Resize
- Compression
- Download
- Image Engine
- Canvas API
- Web Worker
- Face Detection
- Background Replacement
- Batch Processing

---

# Deliverables

Create:

- Base Application Layout
- Responsive Design
- Reusable UI Components
- Theme System
- Navigation Structure

---

# Acceptance Criteria

- Responsive on Desktop, Tablet, and Mobile.
- Light/Dark mode works correctly.
- UI follows the Design System.
- Components are reusable.
- No business logic exists inside UI components.
- No image processing has been implemented.

---

After completing this phase, the application should visually resemble a production-ready product, but all image processing features must remain unimplemented.
